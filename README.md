**Overview**
This app is designed to help nursery owners and staff manage plants, buyers, stock, visitors, and feedback efficiently.
It also includes automated PDF invoice generation for plant purchases.
Built using Java + XML in Android Studio.

**Core Features / Modules**
1. Plant Recognition Module
Identifies plant varieties using image recognition
Shows detailed plant information

2. Visitor Management Module
Tracks visitors
Stores visit history and timestamps

3. Buyer Management Module
Manages buyer profiles
Stores purchase history & basic communication details

4. Feedback Management Module
Collects and analyses feedback
Helps improve nursery services

5. Fertilization Reminder Module
Sets reminders for fertilization schedules
Helps maintain plant health

6. Stock Management Module
Tracks plant stock levels
Updates stock and variety details
Prevents stock shortages

7. Billing & Invoice Module
Generates professional PDF invoices
Stores billing details
Easy to share or print invoices

8. Plant Information Management
Adds, updates, and maintains plant information
Quick access to plant details for users
